1.2 三个数的中位数
  1. 思路：遍历一遍三个数，同时找出最小值，最大值，和它们三个数的和，输出和减去最大值减去最小值的结果，即为三个数的中位数

伪代码：

  1. 最坏情况和平均情况下我的算法都需要进行4次比较 第二和第三个输入分别和min和max比较

  2. 最坏情况下至少进行4次比较 证明同上(2)

1.3 集合最小覆盖问题

  1. U={1,2,3,4} S1={1,2}, S2={1,3}, S3={2,4} 不妨假设同样大小的Si选择最靠前的那个 按上述算法得出的最小覆盖为{S1,S2,S3} 而真正的最小覆盖为{S2,S3}

  2. 算法： 记录所有数字在S中出现的次数 然后从S中删除 所有元素次数大于1并且集合最小的Si 并把相应元素次数减一 重复上述步骤，直到再也不能删除一个集合，此时的S为一个集合覆盖

正确性证明： 无论怎么删除都保证每个元素在覆盖中至少出现一次，即删除过的S是U的一个覆盖

(3)不能 反例：U={1,2,3,4} S1={1,2} S2={3,4} S3={1,4} S4={2,4} 按我的算法得出的最小覆盖是{S2,S3,S4} 而真正的最小覆盖是{S1,S2}

1.7多项式计算

数学归纳法： 1.n=0时 p=a[0]=a0 显然正确 2.假设n=k时正确，则当n=k+1时 由n=k时正确可得 前k次循环(HORNER(A[1...k+1],x)后 \[ p=a_{k+1}x^{k}+a_{k}x^{k-1}...+a_{1} \] 第k+1次循环p=p*x+A[0] 得 \[ p=a_{k+1}x^{k+1}+a_{k}x^{k}...+a_{1}x+a_{0} \] n=k+1时得证 3.由数学归纳法可知算法正确

1.8整数相乘

(1)数学归纳法： 1.z=0时，返回值为0 显然成立 2.假设z=k时算法正确，则当z=k+1时，返回值为 \[ \begin{array} INT\_MULT(2\cdot y,\lfloor \frac{k+1}{2} \rfloor)+y\cdot((k+1)\ mod\ 2)\\ 因为\lfloor \frac{k+1}{2} \rfloor \leq k，所以INT\_MULT函数返回值正确\\ 如果(k+1)mod\ 2为0，则返回值为INT\_MULT(2\cdot y,\frac{k+1}{2})等于y\cdot(k+1)即y\cdot z\\ 如果(k+1)mod\ 2为1，则返回值为INT\_MULT(2\cdot y,\frac{k}{2})+y等于y\cdot k+y\cdot k即y\cdot z\\ z=k+1时得证 \end{array} \] 3.由数学归纳法可知算法正确

(2)不妨设 \[ z/c=p \ \ \ \ z\ mod\ c =q\ 即z=cp+q \] 数学归纳法： 1.z=0时，返回值为0 显然成立 2.假设z=k时算法正确，则当z=k+1时，返回值为 \[ INT\_MULT(c\cdot y,\lfloor \frac{k+1}{c} \rfloor)+y\cdot((k+1)\ mod\ c) \] \[ 因为 c\geq 2\ \ 则\lfloor \frac{k+1}{c} \rfloor \leq k，所以INT\_MULT函数返回值正确 \] \[ =INT\_MULT(c\cdot y,p)+y\cdot q \] \[ =cyp+yq=y(cp+q)=yz \] z=k+1时得证 3.由数学归纳法可知算法正确

2.7函数渐近增长率的基本性质
  1. \[ O:任意f(n)=O(g(n)),\ g(n)=O(h(n)) \]

\[ 则\lim_{n \to \infty}\frac{f(n)}{g(n)}=c1<\infty,\ \lim_{n \to \infty}\frac{g(n)}{h(n)}=c2<\infty \]

\[ \lim_{n \to \infty}\frac{f(n)}{h(n)}=c1\cdot c2 <\infty \]

\[ 所以f(n)=O(h(n))，即O满足传递性 \]

\[ \Omega:任意f(n)=\Omega(g(n)),\ g(n)=\Omega(h(n)) \]

\[ 则\lim_{n \to \infty}\frac{f(n)}{g(n)}=c1>0,\ \lim_{n \to \infty}\frac{g(n)}{h(n)}=c2>0 \]

\[ \lim_{n \to \infty}\frac{f(n)}{h(n)}=c1\cdot c2>0 \]

\[ 所以f(n)=\Omega(h(n))，即\Omega满足传递性 \] \[ \Theta:任意f(n)=\Theta(g(n)),\ g(n)=\Theta(h(n)) \]

\[ 则\lim_{n \to \infty}\frac{f(n)}{g(n)}=c1,\ \lim_{n \to \infty}\frac{g(n)}{h(n)}=c2,\ 0\le c1,c2\le \infty \]

\[ \lim_{n \to \infty}\frac{f(n)}{h(n)}=c1\cdot c2,\ 0\le c1\cdot c2 \le \infty \]

\[ 所以f(n)=\Theta(h(n))，即\Theta满足传递性 \] \[ o:任意f(n)=o(g(n)),\ g(n)=o(h(n)) \]

\[ 则\lim_{n \to \infty}\frac{f(n)}{g(n)}=0,\ \lim_{n \to \infty}\frac{g(n)}{h(n)}=0 \]

\[ \lim_{n \to \infty}\frac{f(n)}{h(n)}=0 \]

\[ 所以f(n)=o(h(n))，即o满足传递性 \] \[ \omega:任意f(n)=\omega(g(n)),\ g(n)=\omega(h(n)) \]

\[ 则\lim_{n \to \infty}\frac{f(n)}{g(n)}=\infty,\ \lim_{n \to \infty}\frac{g(n)}{h(n)}=\infty \]

\[ \lim_{n \to \infty}\frac{f(n)}{h(n)}=\infty \]

\[ 所以f(n)=\omega(h(n))，即\omega满足传递性 \]

  1. \[ O:\lim_{n \to \infty}\frac{f(n)}{f(n)}=1<\infty \]

\[ f(n)=O(f(n))\ 即O满足自反性 \]

\[ \Omega:\lim_{n \to \infty}\frac{f(n)}{f(n)}=1>0 \]

\[ f(n)=\Omega(f(n))\ 即\Omega满足自反性 \]

\[ \Theta:\lim_{n \to \infty}\frac{f(n)}{f(n)}=1,\ 0<1<\infty \]

\[ f(n)=\Theta(f(n))\ 即\Theta满足自反性 \] (3) \[ 由(1)(2)知，\Theta满足传递性和自反性，要证\Theta是一个等价关系，只要证\Theta满足对称性 \]

\[ 对于任意f(n)=\Theta(g(n)),\ \lim_{n \to \infty}\frac{f(n)}{g(n)}=c,\ 0<c<\infty \]

\[ \lim_{n \to \infty}\frac{g(n)}{f(n)}=\frac{1}{c},\ 0<\frac{1}{c}<\infty,\ 即g(n)=\Theta(f(n)),\ \Theta有对称性，原命题得证 \]

  1. \[ f(n)=\Theta(g(n))\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \]

  1. \[ f=o(g)\Leftrightarrow lim\frac{f}{g}=c<\infty\Leftrightarrow lim\frac{g}{f}=\frac{1}{c} >0\Leftrightarrow g=\Omega(f) \]

\[ f=o(g)\Leftrightarrow lim\frac{f}{g}=0\Leftrightarrow lim\frac{g}{f}=\infty\Leftrightarrow g=\omega(f) \]

  1. \[ f=o(g),\ h=\omega(g)\Rightarrow lim\frac{f}{g}=0, \ lim\frac{h}{g}=\infty\Rightarrow f\cap h=\emptyset \]

3.2冒泡排序

(1)数学归纳法 1.n=2,ifA[0]>A[1] then SWAP(A[0],A[1])，则A[1]>=A[0],显然正确 2.假设n=k时算法正确，则当n=k+1时 \[ i:=k+1，j:=1\ to\ k, if\ A[j]>A[j+1] \ Then\ SWAP,\ 使得A[k]=\max_{0\le i \le k}A[i]\\ i:=k\ to\ 2，由n=k时算法正确可得A[0]\le A[1]...\le A[k-1],\\ 综合i=k+1时的循环可得A[0]\le A[1]...\le A[k-1]\le A[k], 所以n=k+1时算法正确 \] 3.由数学归纳法可得，冒泡排序正确

(2)最坏情况和平均情况都是 \[ \sum_{i=2}^{n}\sum_{j=1}^{i-1}1=\frac{n(n-1)}{2}---O(n^2) \] (3) \[ \begin{array} 最坏情况原数组逆序，改进不影响比较次数，所以不会影响最坏情况的时间复杂度\\ 平均情况比较次数会减少，但是时间复杂度仍然是O(n^2) \end{array} \]

3.5

正确性证明： \[ \begin{array} 原算法正确的前提下，循环中j=j-1向前遍历，如果j-1>P[j],则A[j-1]\le A[j]\le A[i],\\ 直到j-1=P[j],\ 才有可能A[j-1]>A[i],所以该算法肯定正确。 \end{array} \] 时间复杂度计算： \[ \begin{array} 设计算P[i]的比较次数为k_{i}，此时P[1..i-1]已经计算完成\\ 若A[i-1]>A[i],则k_{i}=1\\ 若A[i-1]\le A[i],则p[i]一定在p[0..i-1]中，则k_{i}>1,p[i]=p[i-k_{i}+1]\\ 同理有k_{i+1}=1或k_{i+1}>1且有关系k_{i+1}\le i-k_{i}+2,即k_{i}+k_{i+1}<=i+2\\ 因为p[i+1]一定在p[1..i]中，p[i]=p[i-k_{i}+1]\\ 同理k_{i+m}=1，或k_{i+m}\le i-(k_{i}+...+k_{i+m})+m+1,即k_{i}+...+k_{i+m}\le i+m+1\\ 令i=2，m=n-2,则k_{2}+...+k_{n}\le n+1\\ 又k1=0,所以复杂度为\Theta(n) \end{array} \]

3.8微博名人问题

(1)可能有0或1个名人

(2)思路：一开始所有人都可能是名人，然后对他们进行遍历，如果关注了别人，那么自己不是名人，如果没关注那个人那么那个人不是名人，如果这个人没有关注任何人，则对其他人进行遍历，如果其他人都关注了他，那么他是名人，如果有人没有关注他，那么所有人中没有名人。

3.9最大和连续子序列

(1)遍历所有子串并求和

(2)求和的时候用累加

(3)将数组平分，分别求出求出连续子序列的最大和，同时从中间元素开始分别向左和向右算出最大的和并相加，三个最大和中最大的就是整个数组中连续子序列的最大和

(4)遍历同时求和，如果和小于0就舍弃，记录过程中的最大和即为结果

(5)dp[n]：以n为结尾的连续子序列的最大和，以dp[i] := MAX(0,dp[i-1])+A[i]进行动态规划，答案为MAX(dp[i])
